# 公约
## 1.规则&&提交规范
- 周一到周五，每日一道算法题！
- 周末回顾总结！
- 小组成员每个月至少参与刷20道题目（用自己熟悉的语言提供好的解法即可）
- 代码提交github仓库，必须保证以下几点
    - 代码完整、能正常运行
    - 注释自己的解题思路
    - 附带提交运行时间作为参考

## 题型
|工作日|题型|
|--|--|
|周一|2道简单|
|周二|1道中等|
|周三|1道中等|
|周四|1道困难|
|周五|1道创新题|

## 2.奖惩机制
- 每晚10点30之前未提交代码，或代码执行不能运行通过的组员需要发10元红包到群里（组长每日检查）
- 红包作为刷题基金，暂由豪杰保管
- 11月05日，大家讲出自己想买的技术学习资料，豪杰用刷题基金统一购买
- ⚠️连续3个月低于20道/月的队员移除组织⚠️
