## 题目描述

```
在未排序的数组中找到第 k 个最大的元素。请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

示例 1:

输入: [3,2,1,5,6,4] 和 k = 2
输出: 5
示例 2:

输入: [3,2,3,1,2,4,5,5,6] 和 k = 4
输出: 4
说明:

你可以假设 k 总是有效的，且 1 ≤ k ≤ 数组的长度。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/kth-largest-element-in-an-array
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```

## 思路
**暴力求解**
- 先从大到小进行排序
- 返回第 k 个元素

**Heap**
- 创建一个大顶堆
- 遍历数组将每一个元素依次入栈
- 保持堆的大小小于等于 k
- 返回堆顶的元素

## 关键点解析

- 堆
- 分治

```java
class Solution {
    public int findKthLargest(int[] nums, int k) {
        // 1、把无序数组构建成最小堆
        for (int i = (nums.length - 2)/2;i >= 0;i--) {
            downAdjust(nums,i,nums.length);
        }
        // 2、循环删除堆顶元素，移到集合尾部，调整堆产生新的堆顶
        for (int i = nums.length - 1; i > 0; i--) {
            // 最后一个元素和第一个元素进行交换
            int temp = nums[i];
            nums[i] = nums[0];
            nums[0] = temp;
            // 调整为最小堆
            downAdjust(nums,0,i);
        }
        return nums[k-1];
    }

     /**
     * "上浮"调整（让小的数据上浮，构建最小堆）
     * @param array     待调整的堆
     * @param parentIndex       要上浮的父节点
     * @param length    堆的有效大小
     */
    public void downAdjust(int[] array,int parentIndex,int length){
        // temp保存父节点，用于最后的复制
        int temp = array[parentIndex];
        int childIndex = 2 * parentIndex + 1;
        while (childIndex < length) {
            // 如果有右孩子，且右孩子小于左孩子，则定位到右孩子
            if (childIndex + 1 <length && array[childIndex + 1] < array[childIndex]) {
                childIndex++;
            }
            // 如果父节点小于任何一个孩子节点，则直接跳出
            if (temp <= array[childIndex]) {
                break;
            }
            // 无需真正交换，单向赋值即可
            array[parentIndex] = array[childIndex];
            parentIndex = childIndex;
            childIndex = 2 * childIndex + 1;
        }
        // 父节点上浮后，赋值
        array[parentIndex] = temp;
    }
}
```

function swap(arr, i, j) { // 用于互换元素位置
  let temp = arr[i];
  arr[i] = arr[j];
  arr[j] = temp;
}
function heapify(arr, x, length) {
  let l = 2 * x + 1; // ❓
  let r = 2 * x + 2; // ❓
  let largest = x;
  if (l < length && arr[l] > arr[largest]) {
    largest = l;
  }
  if (r < length && arr[r] > arr[largest]) {
    largest = r;
  }
  if (largest != x) {
    swap(arr, x, largest); // 位置交换
    heapify(arr, largest, length); // 递归交换以下是否建好堆
  }
}
var findKthLargest = function(nums, k) {
  let size = nums.length;
  // 建立堆
  for (let i = parseInt(size / 2) + 1; i >= 0; i--) {
    heapify(nums, i, size);
  }
  // 排序
  for (let j = size - 1; j >= size - k; j--) {
    // 得到本次的最大，将最大的与最后一个交换位子
    swap(nums, 0, j);
    heapify(nums, 0, j);
  }
  return nums[size - k];
};
